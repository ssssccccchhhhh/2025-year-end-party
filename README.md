## 📋 이벤트 안내

### Event 1 - Mad Max 💨
- **AMRAP 3min**: 15 Sync Burpee Box Jump + Max Cal Row/Ski/Bike
- **참가 인원**: 여자 3명 (3분) + 남자 3명 (3분) = 총 6명
- **포맷**: 여자 그룹 먼저 3분, 1분 브레이크, 남자 그룹 3분

### Event 2 - Triple Lift 🏋️
- **종목**: 1RM Hang Snatch, 1RM Hang Clean, Max Thruster
- **참가 인원**: 여자 3명 (각 종목별 1명) + 남자 3명 (각 종목별 1명) = 총 6명
- **포맷**: 스내치 선수, 클린 선수, 쓰러스터 선수 각각 따로 측정

### Event 3 - Skyscraper 🧗
- **AMRAP 3min**: 5 Rope Climb + Max Pull-up 변형
- **참가 인원**: 남자 3명 (Ring MU, Bar MU, C2B) 또는 여자 1명 (Pull-up)
- **포맷**: 3분 안에 로프 5개 완료 후 남은 시간 풀업
- **출전 (변경)**:
  - 팀 A/D: 남자 체투바 + 여자 풀업 출전
  - 팀 B/C: 여자 2명 풀업/풀업 출전
- **채점 기준 (변경)**:
  - 링머슬업 + 바머슬업 갯수로 4,3,2,1점
  - 체투바(풀업) + 풀업 갯수로 4,3,2,1점

### Event 4 - Super Train 🚂
- **종목**: 6 Rounds Relay (200m Run + 50m Sled Push 왕복2번)
- **참가 인원**: 남녀 혼합 6명 (릴레이)
- **포맷**: 6명이 순서대로 릴레이 형식으로 진행

### Event 5 - Tug of Sled ⚔️
- **종목**: 1330파운드(605kg) Sled Pull
- **참가 인원**: 남자 3명
- **포맷**: 2분 동안 최대한 멀리 끌고 가기

### Event 6 - Road to Strength 💪
- **종목**: 1min Max Seated Row, Chest Press, Leg Press
- **참가 인원**: 여자 3명 (각 머신별 1명) + 남자 3명 (각 머신별 1명) = 총 6명
- **포맷**: 각 머신마다 1분씩 WORK 값 측정

## 📱 사용 방법

### 1️⃣ 팀 선택
상단에서 자신의 팀(A/B/C/D)을 선택하세요.

### 2️⃣ 코치 확인
각 팀의 담당 코치를 확인하세요.
- 재경, 봉수, 세영, 예지 코치님

### 3️⃣ 이벤트 규칙 확인
- 각 이벤트 카드의 **상단 영역(이벤트 제목 부분)을 클릭**하면
- 📖 **상세 룰북**이 아코디언 형태로 펼쳐집니다
- 운동 방법, 기록 방식, 세부 규칙을 확인할 수 있어요
- 다시 클릭하면 접을 수 있습니다

### 4️⃣ 선수 배정
각 이벤트 카드에서:
- 원하는 슬롯을 클릭하면 선수 목록이 나타납니다
- 참가할 선수를 선택하면 자동으로 배정됩니다
- **주황색 테두리** 슬롯은 코치도 배정 가능한 자리입니다

### 5️⃣ 배정 확인
- **멤버 리스트**에서 각 선수가 몇 개 이벤트에 배정되었는지 확인
- ⚠️ **붉은색 표시**: 한 선수가 너무 많은 이벤트에 배정된 경우 (재분배 권장)
- 각 이벤트별로 누가 배정되었는지 한눈에 확인

### 6️⃣ 배정표 저장 & 공유
- 하단 **"📸 배정표 캡처"** 버튼을 눌러 이미지로 저장
- **"📖 룰북 캡처"** 버튼으로 전체 이벤트 규칙을 이미지로 저장
- **"🔗 URL 복사"** 버튼으로 현재 배정 상태가 담긴 URL을 복사
- 팀 단톡방에 공유하세요!
